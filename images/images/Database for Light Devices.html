Here's an example code snippet that listens for signals sent over the serial port and turns on or off a light based on the content of the signal:

arduino
// define the pin numbers for the lights
const int LIGHT_1_PIN = 2;
const int LIGHT_2_PIN = 3;
const int LIGHT_3_PIN = 4;
const int LIGHT_4_PIN = 5;

// set up the serial communication
void setup() {
  Serial.begin(9600);   // set baud rate to 9600
  pinMode(LIGHT_1_PIN, OUTPUT);
  pinMode(LIGHT_2_PIN, OUTPUT);
  pinMode(LIGHT_3_PIN, OUTPUT);
  pinMode(LIGHT_4_PIN, OUTPUT);
}

// listen for incoming signals and take action based on the content
void loop() {
  if (Serial.available() > 0) {
    // read the incoming signal
    String signal = Serial.readStringUntil('\n');
    // parse the signal and take action based on the content
    if (signal.startsWith("on L1")) {
      digitalWrite(LIGHT_1_PIN, HIGH);
      Serial.println("Light 1 turned on");
    } else if (signal.startsWith("off L1")) {
      digitalWrite(LIGHT_1_PIN, LOW);
      Serial.println("Light 1 turned off");
    } else if (signal.startsWith("on L2")) {
      digitalWrite(LIGHT_2_PIN, HIGH);
      Serial.println("Light 2 turned on");
    } else if (signal.startsWith("off L2")) {
      digitalWrite(LIGHT_2_PIN, LOW);
      Serial.println("Light 2 turned off");
    } else if (signal.startsWith("on L3")) {
      digitalWrite(LIGHT_3_PIN, HIGH);
      Serial.println("Light 3 turned on");
    } else if (signal.startsWith("off L3")) {
      digitalWrite(LIGHT_3_PIN, LOW);
      Serial.println("Light 3 turned off");
    } else if (signal.startsWith("on L4")) {
      digitalWrite(LIGHT_4_PIN, HIGH);
      Serial.println("Light 4 turned on");
    } else if (signal.startsWith("off L4")) {
      digitalWrite(LIGHT_4_PIN, LOW);
      Serial.println("Light 4 turned off");
    }
  }
}

In this example, the Serial library is used to establish a serial communication channel between the Arduino and the Node.js web app. The setup() function is used to set up the pins for the lights as output pins and to initialize the serial communication with a baud rate of 9600. The loop() function is used to listen for incoming signals over the serial port and take action based on the content of the signal. When a signal is received, the Arduino reads the signal, parses the content, and uses the digitalWrite() function to turn on or off the corresponding light based on the content of the signal.

You can modify this code to include logic for controlling other devices or for responding to different types of signals based on your specific project requirements.

arduino
// define the pin numbers for the lights
const int LIGHT_1_PIN = 2;
const int LIGHT_2_PIN = 3;
const int LIGHT_3_PIN = 4;
const int LIGHT_4_PIN = 5;

// define an array of device IDs
const String DEVICE_IDS[] = {"L1", "L2", "L3", "L4"};

// set up the serial communication
void setup() {
  Serial.begin(9600);   // set baud rate to 9600
  pinMode(LIGHT_1_PIN, OUTPUT);
  pinMode(LIGHT_2_PIN, OUTPUT);
  pinMode(LIGHT_3_PIN, OUTPUT);
  pinMode(LIGHT_4_PIN, OUTPUT);
}

// listen for incoming signals and take action based on the content
void loop() {
  if (Serial.available() > 0) {
    // read the incoming signal
    String signal = Serial.readStringUntil('\n');
    // loop through the device IDs and check if the signal matches any of them
    for (int i = 0; i < 4; i++) {
      if (signal.startsWith("on " + DEVICE_IDS[i])) {
        digitalWrite(i + 2, HIGH);  // turn on the corresponding light
        Serial.println("Light " + DEVICE_IDS[i] + " turned on");
        break;  // exit the loop once a match is found
      } else if (signal.startsWith("off " + DEVICE_IDS[i])) {
        digitalWrite(i + 2, LOW);  // turn off the corresponding light
        Serial.println("Light " + DEVICE_IDS[i] + " turned off");
        break;  // exit the loop once a match is found
      }
    }
  }
}

In this example, the DEVICE_IDS array is used to store the IDs for each device that can be controlled by the incoming signal. The loop in the loop() function checks the incoming signal against each ID in the array and takes action based on the content of the signal. When a match is found, the Arduino uses the digitalWrite() function to turn on or off the corresponding light and prints a message over the serial port to confirm the action that was taken.
